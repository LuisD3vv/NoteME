La cardinalidad

se define como la relaciones entre entidades (tablas) con otro conjunto, se pueden compartir solo elementos únicos, o varios a muchas tablas, a eso se le llama cardinalidad : puede haber de 3 tipos principalmente 1-Uno a Uno 2-Uno a muchos 3-Muchos a muchos (se requiere de una tabla intermedia)
